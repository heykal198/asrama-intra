Cara Menutup Pop-Up Blocker di Pelayan Web
Bagaimana Untuk Menutup Pop-Up Blocker Internet Explorer.
    Pilih Tools, kemudian pilih Internet Options
   
    Pilih tab Privacy, kemudian untick Turn on Pop-up Blocker dan klik "OK".
   
Bagaimana Untuk Menutup Pop-Up Blocker Mozilla Firefox.
    Pilih Tools, kemudian pilih Options
   
    Pilih tab Content, kemudian untick Block pop-up windows dan klik "OK".
